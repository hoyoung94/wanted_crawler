## 핵심 역량 요약

### 1. 클라우드 운영 환경 이해

한국폴리텍대학 클라우드서비스보안 하이테크 과정과 한국능률협회 AWS 클라우드 컴퓨팅 전문가 양성과정을 통해 클라우드는 단순히 자원을 배치하는 기술이 아니라 계정, 권한, 네트워크, 보안, 운영 안정성을 함께 관리해야 하는 운영 환경이라는 점을 학습했다.

### 2. 운영을 고려한 백엔드 구조 설계

(주)린에이아이 인턴 당시 음식점용 QR 메뉴판 및 챗봇서비스 MUMUL 프로젝트에 참여해 Django 기반 Backend와 Admin Page 기능을 개발했다. 회원, 업장, 메뉴, 메뉴 이미지, 게시판 데이터를 분리하고 업장 단위로 연결되는 구조를 고려해 관리자 페이지에서 필요한 정보를 조회·수정할 수 있도록 구현했다.

### 3. 문제 원인 분류와 사용자 대응

한국방송통신대학교 서비스센터에서 원격지원 및 프로그램 오류 해결 업무를 수행하며 PC, 모바일, 프로그램 환경을 구분해 문제 조건을 파악하고 사용자에게 원인과 조치 방향을 설명하는 경험을 쌓았다.

### 4. 데이터 기반 문제 개선

AWS 기반 서울시 모기 활동 지수 예측 웹서비스 프로젝트에서 데이터 수집, 전처리, 모델 테스트, 분석을 담당했다. 기상데이터 외에 모기 개체수와 종별 분류 결과를 함께 반영해 변수 조합을 개선했고, 최종 결정계수 0.63을 달성했다.

- 서비스가 안정적으로 유지되도록 구축 이후에도 지속적으로 점검하고 문제에 대응하는 엔지니어를 지향함
- Linux와 AWS 환경에 대한 이해를 바탕으로 운영 흐름에 빠르게 적응하고자 함

활용 문장:

> 클라우드는 단순히 자원을 배치하는 기술이 아니라 계정과 권한, 네트워크, 보안, 운영 안정성을 함께 관리해야 하는 운영 환경이라고 이해하고 있습니다. 저는 서비스의 이상 징후를 먼저 파악하고, 로그와 상태 정보를 바탕으로 원인을 단계적으로 분석하며 안정적인 운영을 지원하는 엔지니어로 성장하고자 합니다.

- Django 기반 Backend 및 Admin Page 개발

- 관리자 페이지에서 필요한 정보를 쉽게 조회·수정할 수 있도록 운영 편의성 고려

활용 문장:

- 사용자에게 원인과 조치 방향을 이해하기 쉽게 설명
